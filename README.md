
# FastAPI with Postgres
- FastAPI app
- PostgreSQL with SQLAlchemy
- Python-Decouple
- Pydantic

## Docker
```docker compose up --build```

## Run
```poetry run uvicorn api.main:app --reload```


## Postgres
Run postgres: ```sudo -u postgres psql```

Show databases: ```\l```

Use databases: ```\c```

Show tables: ```\dt```

Restart postgres server: ```sudo service postgresql restart```
