
# FastAPI with Postgres
- FastAPI app
- PostgreSQL with SQLAlchemy
- Python-Decouple
- Pydantic

## Docker
```Hello user$ docker-compose up --build```

## Run
```poetry run uvicorn api.main:app --reload```


## Postgres
Run postgres: ```Sudo -u postgres psql```

Show databases: ```\l```

Use databases: ```\c```

Show tables: ```\dt```
